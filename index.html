My Toolkit

My Favorite Frameworks

  1. Django is a high-level framework that promotes rapid development and clean, pragmatic design. It includes everything needed to build a web app, like ORM, admin panel, authentication, and more, following the “batteries-included” philosophy.
  2. Node.js allows JavaScript to run on the server-side. It uses an event-driven, non-blocking I/O model, making it efficient and scalable. It's commonly used with frameworks like Express.js to build backend services.
  3. Vite is a fast build tool and development server for modern frontend projects. It offers instant server start and lightning-fast hot module replacement, primarily used with frameworks like Vue and React.
  4. Flask is a micro-framework that gives developers the essentials to build web applications. It’s minimal, flexible, and unopinionated, making it ideal for small to medium-sized projects where customization is needed.
  5. Next.js is a full-stack framework built on React, enabling server-side rendering, static site generation, and API routes. It's great for performance-optimized, SEO-friendly web apps.
  6. Ruby on Rails (or just Rails) is a convention-over-configuration framework that helps developers build applications quickly. It includes built-in tools for routing, database migrations, testing, and more.
